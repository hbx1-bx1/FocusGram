# FocusGram Desktop

Electron-based Instagram DM-only wrapper.

## Quick start

```
npm install
npm run dev
```

## Build

```
npm run build:win
npm run build:mac
npm run build:linux
```

Output goes to `dist/`.

## Features

- Opens Instagram DMs in a dedicated window
- Blocks navigation to Reels, Explore, Posts, Stories, Profiles
- Clears session data on demand
- Keyboard shortcuts for Reload, Back, Forward
